# 3DGFX-Gradle

----

## Installation

- Windows:

  ```CMD
  git clone https://github.com/NPEX42/3DGFX-Gradle.git
  gradlew build
  ```

- Linux & MacOS:

  ```bash
  git clone https://github.com/NPEX42/3DGFX-Gradle.git
  chmod +x gradlew
  gradlew build
  ```
